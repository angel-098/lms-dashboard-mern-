# LMS Dashboard (MERN)

## Features
- Add & view courses
- Add & view lessons
- Mark lessons as completed
- Dashboard stats

## APIs
- POST /api/courses
- GET /api/courses
- POST /api/lessons
- GET /api/lessons
- PUT /api/lessons/:id
